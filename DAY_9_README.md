 Day 9 - JIRA & Bug Tracking Tools

 Objective
To learn JIRA as a bug tracking tool by creating a project, logging bugs, using workflow transitions, working with Scrum board, creating dashboards, and practicing JQL queries.

Tool Used
- Atlassian JIRA Cloud (Free Version)

 Activities Performed

 JIRA Account Setup
- Created a free Atlassian JIRA Cloud account.
- Explored the JIRA interface and navigation.

 Project Creation
- Created a Scrum project for bug tracking practice.
- Configured workflow statuses:
  - To Do
  - In Progress
  - In Review
  - Done


Understanding JIRA Issue Types
Learned different JIRA issue types used in Agile projects:

- **Bug**: Used to report defects found during testing.
- **Story**: Represents a user requirement.
- **Task**: General work item to be completed.
- **Epic**: Large feature/module that contains multiple stories.
- **Sub-task**: Smaller task under a story or task.

Bug Creation in JIRA
- Logged 20+ bugs using the JIRA issue creation form.
- Filled required fields such as:
  - Summary
  - Description (Steps to Reproduce, Expected vs Actual)
  - Priority
  - Labels
  - Attachments

 Workflow Transitions
Practiced moving bugs through workflow stages:

To Do → In Progress → In Review → Done

Also understood additional statuses like Reopen and Rejected.

Scrum Board and Sprint Practice
- Created Sprint 1.
- Added bugs from backlog to sprint.
- Started sprint and tracked issues using Scrum board.
- Moved issues across board columns based on progress.

JQL Filters and Queries
Created and practiced JQL queries for filtering and tracking bugs.

Example JQL Queries:
- `project = "Bug Tracking Practice" AND issuetype = Bug`
- `project = "Bug Tracking Practice" AND status = "To Do"`
- `project = "Bug Tracking Practice" AND priority = High`
- `project = "Bug Tracking Practice" AND labels = login`
- `project = "Bug Tracking Practice" AND status != Done`

Dashboard Creation
- Created a custom JIRA dashboard.
- Added gadgets for tracking bug metrics such as:
  - Filter Results
  - Pie Chart
  - Issue Statistics
  - Created vs Resolved chart

Deliverables Completed
- JIRA Project Created
- 20+ Bugs Logged in JIRA
- Scrum Board Configured
- Workflow Status Transitions Practiced
- JQL Filter Examples Created
- Custom Dashboard Created

Day 9 helped in gaining hands-on experience with JIRA bug tracking, Agile Scrum workflow, sprint management, dashboards, and reporting using JQL filters.
