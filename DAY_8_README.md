Day 8 - Bug Life Cycle & Bug Reporting

Objective
To understand the bug life cycle process and practice writing effective bug reports with proper severity and priority classification.

Topics Covered
- Bug Life Cycle Stages
- Bug Report Components
- Severity vs Priority
- Best Practices for Bug Reporting

 Bug Life Cycle
New → Assigned → Open → Fixed → Retest → Verified → Closed  
Additional statuses: Reopen, Rejected, Deferred

Bug Reporting Guidelines
A good bug report should contain:
- Bug ID
- Summary
- Description
- Steps to reproduce
- Expected Result
- Actual Result
- Severity and Priority
- Environment details
- Attachments (screenshots/videos)
