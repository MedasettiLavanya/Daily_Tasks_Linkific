Day 5 - White Box Testing & Code Coverage

Concepts Covered
- White Box Testing
- Statement Coverage
- Branch Coverage
- Path Coverage

Key Notes
- Statement coverage ensures every line executes at least once.
- Branch coverage ensures both True and False conditions are tested.
- Path coverage ensures all possible execution paths are tested.

Tools Awareness
- JaCoCo (Java)
- Istanbul (JavaScript)
- Coverage.py (Python)
